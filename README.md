# VenueVista
## Backend Setup
### Install the following
npm install express dotenv mongodb jsonwebtoken bcryptjs express-async-handler cors nodemon

### Create a .env and add the following
PORT = 4000
DB_URL = mongodb://localhost:27017/hallbooking
JWT_SECRET = abcdef

### To run the server
nodemon server.js 

## Frontend Setup
npm install
npm run dev
